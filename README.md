# Sharplingo AI Judge

一个为 **Sharplingo 德语翻译练习页面**设计的浏览器扩展。

它会在练习页面中加入 **“AI评判”** 按钮，读取当前题目的原句、你的答案和参考答案，再调用你自己配置的 AI 模型，对这句德语翻译进行解释式点评。

---

## 这个插件是做什么的

由于德语V2语序的特点和语气词的运用，翻译的正确答案可以有很多形式。

你可能会遇到这些情况：

- 你的答案和参考答案不一样，但意思其实对了
- 语序不同、表达不同，但整体可以接受
- 少了语气词，不知道算不算严重错误
- 你想知道的不是“标准答案是什么”，而是“我这句到底对不对、问题大不大、哪里可以更自然”

这个插件就是为这种场景设计的。

它不是简单比较你和参考答案是否一模一样，而是让 AI 从 **语义、语法、自然度和可接受性** 的角度给出反馈。

---

## 核心功能

- 在 Sharplingo 页面注入 **AI评判** 按钮
- 自动读取：
  - 中文原句
  - 英文原句
  - 用户输入的德语答案
  - 页面中的参考答案
- 支持自定义：
  - API Base URL
  - API Key
  - 模型名
  - 提示词模板
- 支持 **OpenAI 兼容接口**
- 返回结构化点评结果
- 可开启调试日志，方便排查响应慢或接口报错问题

## 安装方法

1. 下载或克隆本仓库
2. 打开浏览器扩展页面：

```text
chrome://extensions/
````

3. 开启右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本项目文件夹

---

## 使用方法

1. 打开 Sharplingo 翻译练习页面
2. 在扩展设置中填入你的：

   * API Base URL
   * API Key
   * 模型名
3. 回到练习页面
4. 输入你的德语答案
5. 点击 **AI评判**
6. 查看模型返回的点评结果

## 模型推荐

“对德语翻译做评判”这类任务，本质上更偏向**语义理解、表达判断和格式化输出**，一般不需要特别强的推理模型。
它是一个长输入短输出的任务，从**性价比**角度来看，目前更推荐：

- `gemini-2.5-flash-lite`这个更便宜
- `gpt-4o-mini`

这两个模型在实际测试中返回速度都比较快，整体体验更适合做句子级别的即时评判。  
按常见使用情况估算，**1 块钱大约可以评判 1000～1500 个句子**，具体会受到句子长短、提示词长度和返回内容长度的影响。

如果你想用更强的模型，也完全可以自己配置，比如 `claude-opus-4-6`（不会真的有人配这个吧）。  
这个插件本身不限制你接入什么模型，按你的需求和预算选择即可。

### 关于其他模型

目前也测试过一些其他模型，例如 `qwen3.5-flash`。  
但在我的使用环境里，它的返回速度明显偏慢，哪怕是很简单的句子，有时也需要 **20–30 秒**。这可能和代理配置、接口兼容层、网络链路，或者模型服务本身的响应速度有关，暂时还不能完全确定是哪一个环节导致的。

后续的一个优化方向，是继续测试并适配更多**价格便宜、速度稳定**的模型。  
不过从当前实际成本来看，这类任务本身已经很便宜了。如果你不想花时间折腾接口和兼容性，直接使用上面推荐的 `gemini-2.5-flash-lite` 或 `gpt-4o-mini` 就够用了。
